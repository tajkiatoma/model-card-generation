**Context Length:**
*   The model supports a maximum position embedding of 4096 tokens (config.json).

## How to Use
This section outlines how to use the model.

The model is a causal language model that can be used for text generation (config.json). It uses a `LlamaTokenizer` (tokenizer_config.json).

**Tokenizer Details:**
*   **Type:** Byte-Pair Encoding (BPE) (tokenizer.json).
*   **Vocabulary Size:** 32,000 (tokenizer.json).
*   **Special Tokens:**
    *   `<s>`: Beginning of sentence (BOS) token, ID: 1 (special_tokens_map.json, tokenizer.json).
    *   `</s>`: End of sentence (EOS) token, ID: 2 (special_tokens_map.json, tokenizer.json).
    *   `<unk>`: Unknown token, ID: 0 (special_tokens_map.json, tokenizer.json).
    *   `<pad>`: Padding token, ID: 32000 (added_tokens.json, generation_config.json).
*   **Input Processing:** The tokenizer prepends a space to the input string before tokenization. The model automatically adds a BOS token (`<s>`) to the beginning of sequences (tokenizer.json).

**Generation Settings:**
The default generation configuration is set to:
*   `do_sample`: `true` (generation_config.json)
*   `temperature`: 0.9 (generation_config.json)
*   `top_p`: 0.6 (generation_config.json)
