## Model Details

The model is identified as Llama 2 (LICENSE.txt; USE_POLICY.md.txt). The model configuration was generated with `transformers` version `4.31.0.dev0` (config.json.txt).

### Model type:
The model is a LlamaForCausalLM, a type of large language model designed for causal language modeling (text generation) (config.json.txt).

**Architecture Details:**
*   **Model Architecture:** `llama` (config.json.txt)
*   **Activation Function:** `silu` (config.json.txt)
*   **Hidden Size:** 4096 (config.json.txt)
*   **Intermediate Size:** 11008 (config.json.txt)
*   **Number of Hidden Layers:** 32 (config.json.txt)
*   **Number of Attention Heads:** 32 (config.json.txt)
*   **Number of Key-Value Heads:** 32 (config.json.txt)
*   **RMS Norm Epsilon:** 1e-05 (config.json.txt)
*   **Initializer Range:** 0.02 (config.json.txt)
*   **Vocabulary Size:** 32000 (config.json.txt)

**Size and Context Length:**
*   **Model Size (Disk):** The total size of the model weights is 13,476,835,328 bytes (approximately 13.48 GB) in SafeTensors format (model.safetensors.index.json.txt) and 26,953,670,656 bytes (approximately 26.95 GB) in PyTorch bin format (pytorch_model.bin.index.json.txt).
*   **Supported Context Length:** The model supports a maximum position embedding of 4096 tokens (config.json.txt).

## How to Use
This section outlines how to use the model.

The model is a causal language model, which takes a sequence of text as input and generates subsequent text. The following configuration parameters are specified for generation:

**Generation Settings:**
*   `do_sample`: true (generation_config.json.txt)
*   `temperature`: 0.9 (generation_config.json.txt)
*   `top_p`: 0.6 (generation_config.json.txt)

**Special Token IDs:**
*   `bos_token_id` (Beginning of Sequence): 1 (config.json.txt; generation_config.json.txt)
*   `eos_token_id` (End of Sequence): 2 (config.json.txt; generation_config.json.txt)
*   `pad_token_id` (Padding): 0 (config.json.txt) or 32000 (generation_config.json.txt). Note the discrepancy between the two configuration files.
